# Easy Effects Convoler

## IRS
- Open Easy Effects app
- Go to ```Output``` > ```Add Effect``` > ```Convoler```
- Click on ```Impulses``` and select ```import impulse```
- Select all files in 2 folders in the [IRS](IRS) folder
- Choose the impulse you like best (I recommend ```XHR SonicFidelity```)

## Preset
- Open Easy Effects app
- Go to ```Output``` > ```Preset```(Under Left Corner)
- Click on import preset button (The button in the feild to type the new preset name)
- Select all the files in [Preset](Preset) folder
- Choose the preset you like best
